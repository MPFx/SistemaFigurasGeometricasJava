# Sistema de Figuras Geométricas (Java)

## Versión
**1.0.0**

## Descripción
Sistema que calcula áreas y perímetros de diferentes figuras geométricas (Círculo, Rectángulo, Triángulo, Cuadrado, Elipse, Trapecio) demostrando herencia y polimorfismo.

## Alcance

### ✅ Qué hace
- Calcular área de diferentes figuras geométricas
- Calcular perímetro de diferentes figuras geométricas
- Gestionar figuras en un lienzo
- Calcular área total de todas las figuras
- Encontrar la figura con mayor área
- Demostrar polimorfismo con lista de tipo `Figura`

### ❌ Qué NO hace
- No tiene interfaz gráfica (solo consola)
- No dibuja las figuras gráficamente

## Tipos de Figura

| Tipo | Descripción | Fórmula Área | Fórmula Perímetro |
|------|-------------|--------------|-------------------|
| CÍRCULO | Figura circular | π × r² | 2 × π × r |
| RECTÁNGULO | Figura rectangular | ancho × alto | 2 × (ancho + alto) |
| TRIÁNGULO | Figura triangular | (base × altura)/2 | ladoA + ladoB + ladoC |
| CUADRADO | Figura cuadrada | lado² | 4 × lado |
| ELIPSE | Figura elíptica | π × a × b | Fórmula de Ramanujan |
| TRAPECIO | Figura trapezoidal | ((B + b) × h)/2 | B + b + L1 + L2 |

| Reporte | Descripción |
|---------|-------------|
| Área Total | Suma de todas las áreas en el lienzo |
| Figura Mayor | Figura con mayor área en el lienzo |
| Comparativa | Comparación entre dos figuras |
